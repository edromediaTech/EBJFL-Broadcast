Voici la mise à jour de la feuille de route pour intégrer la synchronisation Cloud et la gestion de base de données avancée.☁️ 1. Architecture de Synchronisation (Hybrid Cloud)L'application fonctionnera en mode "Offline-First" :Préparation (Cloud) : Sur ebjfl.org, les responsables planifient le culte (liste des chants, versets, noms des intervenants).Synchronisation : Au lancement, EBJFL-Broadcast télécharge le "Programme du Jour" via une API sécurisée.Exécution (Local) : Pendant le service, tout tourne en local (zéro latence, pas de risque de coupure internet).Archivage : À la fin, le rapport du service (chants réellement chantés, logs) est renvoyé sur GCP.🗄️ 2. Structure de la Base de Données (SQLite Locale)Pour garantir une vitesse maximale, l'application embarque une base de données robuste capable de stocker des milliers de textes.TableContenuSource de SynchronisationBiblesVersions LSG, KJV, Creole (Indexées pour l'IA).Import initial / Cloud.ChantsChants d'Espérance (Français/Créole) + Chants Contemporains.ebjfl.org (Master List).PlanningOrdre du service, minutage, intervenants.API GCP (ebjfl.org).LowerThirdsNoms, titres, réseaux sociaux des invités.Formulaire Cloud ou Saisie Locale.🚀 3. Mise à jour de la Feuille de Route (Agent Codeur)Phase 6 : Connectivité Cloud (GCP)[ ] Module d'Authentification : Connexion sécurisée de l'app locale à votre instance GCP (OAuth2 ou Clé API).[ ] Sync Engine : Script de synchronisation bi-directionnelle (Pull: Programmation / Push: Logs).[ ] Gestion des Assets : Téléchargement automatique des images/vidéos liées au service depuis Cloud Storage vers le dossier local assets/.Phase 7 : Moteur de Recherche "Multilingue & Multi-source"[ ] Indexation Full-Text : Recherche ultra-rapide dans les "Chants d'Espérance" par numéro ou par titre.[ ] Programmation Temporelle : Interface pour charger le "Service du Dimanche Matin" qui pré-remplit les boutons de raccourcis (les chants et versets prévus apparaissent en haut de liste).📋 4. Flux de Travail Technique (Exemple)Sur le Web (ebjfl.org) : Le secrétaire saisit le programme : "Chant 145 CE, Verset Psaume 23, Intervenant : Pasteur X".Sur PC (EBJFL-Broadcast) : Le technicien clique sur "Actualiser le Service".
